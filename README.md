# Multithread com Python

Códigos implementados como parte do trabalho de conclusão de curso.

## Ambiente utilizado:
- Python v3.9.6;
- Numba v0.56.4;
- NumPy v1.23.5;
- gcc (GCC) v11.2.0

## Exemplo de execução:
- C:
    - gcc serial.c -w -o serial -lm -O3
    - ./serial

- Python:
    - py serial.py
